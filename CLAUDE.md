# CLAUDE.md — Dynasty Empire Project

This file configures Claude Code's behavior. Read completely before writing code.

## Stack
- **Framework**: Next.js 14+ (App Router)
- **Auth**: Clerk
- **DB**: Neon Postgres + Drizzle ORM
- **Payments**: Stripe
- **UI**: shadcn/ui + Tailwind CSS
- **Deploy**: Vercel

## Design Rules
- Read DESIGN.md and DYNASTY-DESIGN.md before generating any UI
- Dynasty color tokens: Gold #C9A84C, Ink #0A0A0A, Panel #181818
- Mobile-first responsive design
- WCAG AA accessibility compliance

## Conventions
- TypeScript strict mode everywhere
- Server components by default, 'use client' only when needed
- All API routes in app/api/ with proper error handling
- Environment variables in .env.local (never commit secrets)
- Git: conventional commits (feat:, fix:, docs:)

## Dynasty Integration
- Every project gets: BUSINESS-SYSTEM.md, REVENUE-MODEL.md, GTM-PLAYBOOK.md
- Backend generates: main.py (FastAPI), Dockerfile, docker-compose.yml
- Generated by Dynasty Launcher (dynasty-launcher.vercel.app)
